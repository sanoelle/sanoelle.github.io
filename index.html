The Soap Factory

Branding

Art Direction

I absolutely love soap. It smells great, makes you feel wonderful, and if it comes in a pretty package you feel like you’ve won the lottery. For this piece, I was aiming to create a carefree vintage-styled logo and brand. As I did this project for fun, I came up with the company. The Soap Factory was a fitting name and I felt as though the slogan Let’s get sudsy! would be lively and memorable.
